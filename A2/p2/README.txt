Open SLN file and compile on x64 bit to run


CONTROLS:
WASD - Move camera
Scroll wheel - Zoom in/out
Arrow keys - rotate model

Select Model: 1. Teapot 2.Bunny
b: enter b value[0.0, 1.0]
y: enter y value[0.0, 1.0]
z: enter alpha value[0.0, 1.0]
x: enter beta value[0.0, 1.0]
l: enter object colour float float flaot
j: rendering silhouette
k: not rendering silhouette


ambient and specular are both off.
please turn off silhouette if you want to change models, because it's to laggy :(, and you may need to press k multiple times :(((
