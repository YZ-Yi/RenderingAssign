interface using keyboard keys + command line

Select Models: 1. Bunny  2. Car  3. Engine  4. Sphere  5. Teapot  6. Vase

o, p: Include Pi or not
u, i: Include D or not
k, l: Include G or not
h, j: Include F or not
-, =: Include denominator of not
t, y: Using Beckmann or GGX/TR to compute D

press key then enter value in command line
r: roughness [0, 1]
c: object color: [0, 1]
v: ambient strength: [0, 1]
b: specular strength: [0, 1]
m: select material's color
